# 심리 효과 기반 전환 최적화 설계 스펙

## 개요

`give-test.html` 결과 화면에 심리 효과를 적용해 givecosystem.com 유료 전환율을 높이고, 관련 미니테스트 2개와 지식 허브 칼럼 2개를 추가한다. 모든 카피는 사실에만 근거한다.

---

## 1. give-test.html 결과 화면 추가 구간

기존 결과 카드 3개(강점/약점/전략) 바로 아래, 공유 버튼 위에 삽입.

### Zone 1 — 진행률 + 잠금 항목
**적용 효과**: 목표 그래디언트, 자이가르닉, 호기심의 틈, 이케아 효과

- "분석 진행률 35%" 프로그레스 바 (기본 진단 = 부분 분석, 사실)
- 유형별 잠금 항목 3개 (부분 텍스트 blur 처리):
  - 🔒 [유형]형의 숨겨진 관계 함정
  - 🔒 거절 시 죄책감이 드는 이유 ~~와 해결법~~
  - 🔒 유형별 경계 설정 스크립트 ~~3가지~~
- "16개 질문에 성실히 답하셨어요 — 심화 분석까지 한 걸음만 더"

### Zone 2 — 가격 3종 카드
**적용 효과**: 앵커링, 미끼 효과, 프레이밍, 무현금 효과

- 상단: "커피 한 잔 값으로 평생 반복되는 관계 패턴을 끊을 수 있다면?"
- 3종 카드 (좌→우):
  | 카드 | 가격 | 역할 |
  |---|---|---|
  | 30일 챌린지 | ₩14,900 | 미끼 (단독 비쌈) |
  | 심화+30일 세트 🏆인기 | ₩17,000 | 앵커 + 추천 |
  | 심화 분석만 | ₩4,900 | 진입점 |
- 하단: "₩4,900 + ₩14,900 = ₩19,800 → 세트로 하면 ₩17,000"
- 각 카드 클릭 → givecosystem.com

### Zone 3 — 손실 회피 CTA
**적용 효과**: 손실 회피, 폰 레스토프 효과, 대비 효과

- 배경: #fff8f8, 테두리: #ffe5e5 (주변 흰 카드와 대비)
- "지금 이 패턴을 모르면 같은 관계에서 같은 상처를 반복합니다"
- 부연: 기본 진단은 유형을 알려줌 / 왜 반복되는지·어떻게 바꾸는지는 심화에서만
- **[내 관계 약점 지금 분석하기 →]** → givecosystem.com
- 하단 서브텍스트: "100% 사실 기반 분석"

### Zone 4 — 관련 테스트 2개
**적용 효과**: 호기심의 틈, 자기 고양 편향, 사이트 체류 시간 증가

- "나는 호구일까?" → hogoo-check.html
- "내 인간관계 위험도 테스트" → relationship-risk.html

### Zone 5 — 지식 허브 무료 링크
**적용 효과**: 보답 기대 효과, 권위 효과

- 기존 칼럼 4개 + 새 칼럼 2개 링크

## 2. hogoo-check.html — 나는 호구일까?

### 스펙
- 12문항, 4지선다 (0~3점)
- 총점 0~36점 → 5가지 결과 유형

### 결과 유형
| 점수 | 유형 | 태그라인 |
|---|---|---|
| 30~36 | 완전 호구형 | "착하다는 말이 칭찬인지 상처인지 모르겠어요" |
| 22~29 | 착한 사람형 | "선 하나만 더 그으면 관계가 달라집니다" |
| 14~21 | 중간 경계형 | "상황에 따라 달라지는 당신의 패턴" |
| 7~13 | 현실주의형 | "이미 잘 하고 있어요, 더 정교하게 만들면 완벽" |
| 0~6 | 철벽형 | "경계는 탁월해요, 관계의 온기도 함께" |

### 결과 화면
- give-test.html과 동일한 레이아웃 (이모지 캐릭터 + 유형명 + 태그라인 + 카드 3개)
- 카드: 당신의 강점 / 주의할 점 / 지금 할 수 있는 것
- 하단 CTA → givecosystem.com (GIVE ID 심화 테스트 연결)

## 3. relationship-risk.html — 내 인간관계 위험도 테스트

### 스펙
- 12문항, 4지선다 (0~3점)
- 총점 0~36점 → 4가지 결과 유형

### 결과 유형
| 점수 | 유형 | 태그라인 |
|---|---|---|
| 27~36 | 고위험 단계 | "지금 당신의 관계에는 즉각적인 변화가 필요합니다" |

### 결과 화면
- hogoo-check.html과 동일한 구조
- 하단 CTA → givecosystem.com
