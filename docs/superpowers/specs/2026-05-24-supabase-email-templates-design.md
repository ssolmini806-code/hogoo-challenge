# Supabase 이메일 템플릿 커스터마이징 — 디자인 스펙

## 개요

Supabase 인증 이메일 4종을 GIVE ECO SYSTEM 브랜드에 맞게 커스터마이징한다.

---

## 디자인 결정 사항

### 스타일: 헤더 블록형 (B)
- 상단 풀-컬러 헤더 블록에 브랜드명 표시
- 흰 본문 배경
- 그린 CTA 버튼

### 색상
| 용도 | 값 |
|---|---|
| 헤더 배경 / 버튼 | `#00896c` (브랜드 그린, 채도 완화) |
| 헤더 로고 텍스트 | `#e8f7f3` |
| 본문 제목 | `#1d1d1f` |
| 본문 텍스트 | `#5a5a62` |
| 푸터 텍스트 | `#bbbbbb` |
| 구분선 / 배경 | `#f0f0f0` |
| 배지 배경 | `#eef8f4` |
| 배지 텍스트 | `#2e9e7e` |

### 로고 / 브랜드
- 표시명: `GIVE ECO SYSTEM`
- 태그라인: `관계의 새로운 기준`

### 문구 톤
따뜻한 코치형 — 응원하고 공감하는 어조. 부담 없이 친근하게.

## 이메일 4종 스펙

### ① 회원가입 확인 (Confirm Signup)
- **제목**: `[GIVE] 거의 다 왔어요! 이메일만 확인해주세요`
- **제목(h1)**: `드디어 시작이네요! 🌱`
- **본문**:
  ```
  안녕하세요!
  호구 탈출 챌린지의 첫 걸음을 내딛으셨군요.
  아래 버튼을 눌러 이메일을 확인하면, 바로 시작할 수 있어요.
  ```
- **CTA 버튼**: `이메일 인증하기`
- **변수**: `{{ .ConfirmationURL }}`
- **푸터**: `링크는 24시간 유효합니다. / 이 메일을 요청하지 않으셨나요? 무시하셔도 됩니다.`

### ② 비밀번호 재설정 (Reset Password)
- **제목**: `[GIVE] 비밀번호 재설정 링크를 보내드렸어요`
- **제목(h1)**: `괜찮아요, 누구나 잊어버려요 😊`
- **본문**:
  ```
  비밀번호 재설정을 요청하셨군요.
  아래 버튼을 눌러 새 비밀번호를 설정해주세요.
  설정 후 다시 챌린지를 이어가실 수 있어요.
  ```
- **CTA 버튼**: `비밀번호 재설정하기`
- **변수**: `{{ .ConfirmationURL }}`
- **푸터**: `링크는 1시간 유효합니다. / 요청하지 않으셨나요? 이 메일을 무시하시면 됩니다.`

### ③ 이메일 변경 확인 (Change Email)
- **제목**: `[GIVE] 새 이메일 주소를 확인해주세요`
- **제목(h1)**: `이메일 주소를 변경하려고 하시는군요`
- **본문**:
  ```
  새 이메일 주소로 변경 요청이 들어왔어요.
  아래 버튼을 눌러 새 주소를 확인해주시면, 변경이 완료됩니다.
  ```
- **CTA 버튼**: `새 이메일 확인하기`
- **변수**: `{{ .ConfirmationURL }}`
- **푸터**: `링크는 24시간 유효합니다. / 요청하지 않으셨나요? 계정 보안을 위해 즉시 문의해주세요.`

### ④ 초대 (Invite User)
- **제목**: `[GIVE] 호구 챌린지에 초대받으셨어요 🎉`
- **제목(h1)**: `당신도 해볼 수 있어요!`
- **본문**:
  ```
  GIVE ECO SYSTEM에 초대받으셨어요.
  7일 동안의 작은 변화가 관계를 바꿀 수 있어요.
  지금 바로 시작해보세요.
  ```
- **CTA 버튼**: `챌린지 시작하기`
- **변수**: `{{ .ConfirmationURL }}`
- **푸터**: `이 초대가 본인에게 온 것이 맞나요? 맞다면 위 버튼을 눌러주세요. / 원하지 않으시면 무시하셔도 됩니다.`

## 구현 방식

### HTML 템플릿 파일 생성
`supabase/templates/` 디렉토리에 4개 파일 생성:
- `confirmation.html`
- `recovery.html`
- `email-change.html`
- `invite.html`

각 파일은 이메일 클라이언트 호환 인라인 스타일 HTML.

### 적용 방법
Supabase 대시보드 → Authentication → Email Templates 에 각 파일 HTML을 붙여넣기.

### 이메일 호환성 고려 사항
- 인라인 스타일만 사용 (Gmail, Outlook 호환)
- Flexbox/Grid 미사용 → table 레이아웃 또는 block 레이아웃
- 외부 폰트 미사용 → system-ui, sans-serif 폴백
- 최대 너비 600px
- 모바일 대응: `max-width: 100%`

---

## 파일 구조 (구현 후)

```
supabase/
└── templates/
    ├── confirmation.html   # 회원가입 확인
    ├── recovery.html       # 비밀번호 재설정
    ├── email-change.html   # 이메일 변경
    └── invite.html         # 초대
```
